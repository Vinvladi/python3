### Цикл while
Цикл while может быть бесконечным, если мы дали неверные инструкции.
```
Циклы в while в python
while условие:
    инструкция 1
    инструкция 2
    ...
    инструкция N
```
```
while <логическое условие>:
    действие 1
    действие 2
    ...
```
### Пример цикл while 
```python
a = 5    # вывод всех чисел от 5 до 1
while a>0:
    print(a, end=' ')
    a -= 1
```
