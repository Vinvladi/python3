# Command if

### Основная конструкция 
if условие:
    блок действий
else:
    блок действий
```
Блок без множественного условия
```
if - elif - ... - else
if условие_1:
    блок действий 
elif
    блок действий
else:
    блок действий
```
Блок с множественным условием (блоки elif и else не обязательные)
```

### Примеры
```python
x = int(input())
if x % 2 == 0:
    print('Четное')
else:
    print('Нечетное')
```
```python
a = 4  # Найти наибольшее из двух чисел (без новой переменной) 
b = 7
if a>=b:
    print(a)
else:
    print(b)

a = 4  # Найти наибольшее из двух чисел (c новой переменной) 
b = 7
m = a
if b>=m:
    m = b
print(m)


```
